Functional Specification – Snuggle

Document Version: 1.0.61
Last Updated: December 2024
Author: Bruno Oliveira

APP SUMMARY

Name: Snuggle – Birthday Manager

Version: 1.0.61

Platform: Android (Flutter)

Languages: 8 languages (Portuguese, English, Spanish, French, German, Chinese, Arabic, Hindi)

IDEA & PURPOSE

Snuggle is a birthday and special date management app that allows users to never forget important dates of friends, family, and relationships. The app offers smart notifications, offline/online synchronization, and sharing via WhatsApp.

TARGET AUDIENCE

  • People who want to strengthen relationships by remembering important dates
  • Families who want to maintain traditions and celebrate together
  • Couples who value milestones and special anniversaries
  • Professionals who want to maintain good personal relationships
  • Anyone who has ever forgotten an important birthday

MAIN FEATURES

Authentication & Setup

  • Google login (Firebase Auth)
  • Automatic language detection
  • Country and phone configuration
  • Offline support

Birthday Management

  • Add birthdays manually
  • Import contacts from phone
  • Organization by month
  • Free limit: 20 birthdays
  • Premium limit: 200 birthdays

Smart Notifications

  • Automatic notifications at 00:01
  • Custom time notifications
  • Auto notifications at 11:00 AM
  • Sound or silent options
  • Offline operation

Synchronization

  • Firebase ↔ Local sync
  • Full offline operation
  • Automatic cloud backup
  • Data recovery
  • Auto sync when online

Sharing & Deeplinks

  • WhatsApp sharing
  • Special invitation links
  • Receive via deeplinks
  • Custom messages by language

Premium System

  • Google Play Billing
  • 200 birthday limit
  • Exclusive dark theme
  • No ads
  • Priority backup

USER FLOW

First Access

Splash Screen → Authentication → Language/Country Setup → Main Screen

Authenticated User

Splash Screen → Main Screen (Birthdays)

Main Navigation

Main Screen ↔ Side Drawer → Add Birthday → Search → Share → Premium

TECHNICAL SPECIFICATIONS

Frontend

  • Flutter 3.0+
  • Dart 2.17+
  • Material Design 3

Backend

  • Firebase Auth
  • Cloud Firestore
  • Cloud Functions

Notifications

  • Local Notifications

Purchases

  • Google Play Billing

COMPATIBILITY

  • Android: 6.0+ (API 23+)
  • Devices: Tablets and smartphones
  • Orientation: Portrait and landscape
  • Operation: Full offline support
  • Size: ~32MB

REQUIRED PERMISSIONS

  • Notifications: To send birthday reminders
  • Contacts: To import birthdays (optional)
  • Internet: For synchronization and backup
  • Alarms: For scheduled notification functionality
